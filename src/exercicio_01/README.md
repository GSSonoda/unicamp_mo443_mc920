# Exercício 01

## Objetivo

Rotacionar uma imagem monocromática em 90 graus no sentido horário, 180 graus e 270 graus no sentido horário, manipulando diretamente os índices da imagem.

## Entradas

- imagem baixada automaticamente do site do professor em `data/input/exercicio_01/`.

## Saídas

- `baboon_rotacao_90_horario.png`;
- `baboon_rotacao_180.png`;
- `baboon_rotacao_270_horario.png`;
- resultados gerados em `results/exercicio_01/`;
- figuras finais do relatório em `docs/relatorio/figuras/exercicio_01/`.

## Código

- `main.py`: baixa a imagem de entrada, executa as rotações e salva as saídas.

## Comandos úteis

Executar a partir da raiz do repositório:

```bash
python3 main.py 1
```
